## Requisitos para rodar o Script
> Não há garantias de que o Script irá funcionar em uma versão diferente do CentOS)

Sistema operacional CentOS7.  
> Para fins de teste, o repositório já vem com uma aplicação PHP

Uma aplicação PHP. 

## Como rodar o Script
Para rodar o script de forma correta, coloque uma aplicação php qualquer dentro do diretório /etc/content e rode o script. Após isso, consulte o endereço 127.0.0.1:80.
> É possível alterar o script a fim de configurar um serviço de DNS. De modo padrão o script cria o seguinte DNS:  
> "IP privado da máquina" -> www."hostname da máquina".com
